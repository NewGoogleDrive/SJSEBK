How to Use

  1. Upload the squeakjs/ folder to your site or GitHub Pages.
  2. Place your .image file inside the images/ folder.
  3. Embed it anywhere with this snippet:
<iframe src="squeakjs/run/index.html#image=../images/YourImage.image" width="800" height="600" allowfullscreen></iframe>

Use relative or absolute paths and adjust the size freely.
